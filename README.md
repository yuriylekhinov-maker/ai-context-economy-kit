# AI Context Economy Kit

Набор навыков, промптов и шаблонов для экономной работы с ИИ над проектами.

Цель: уменьшить расход контекста и цены без заметной потери качества.

## Disclaimer

This project provides workflow templates and Codex skill examples. It does not provide legal, financial, medical, security, or professional advice. Use at your own risk.

## Для кого

- Новички, которые делают первый проект с ИИ.
- Пользователи Codex/AI-агентов, у которых быстро разрастаются чаты.
- Люди, которым нужно продолжать работу в новых чатах без перечитывания всего проекта.

## Что внутри

- `PROMPTS_RU.txt` - 5 стартовых промптов.
- `skills/` - навыки Codex.
- `templates/` - шаблоны рабочих файлов.
- `project-template/` - рекомендуемая структура проекта.
- `AUDIT_CHECKLIST_RU.md` - чеклист аудита расхода контекста.
- `examples/` - примеры проверки на реальных проектах.

## Базовая идея

ИИ не должен читать весь проект по умолчанию. Он должен начинать с компактного актуального контекста, затем искать только нужные места.

Минимальная папка контекста проекта:

```text
00_Context/
  CURRENT_STATE.md
  NEXT_WORK.md
  PROJECT_MAP.md
  DECISIONS.md
```

## Навыки

- `context-economy` - экономная работа с контекстом.
- `handoff-summary` - краткое резюме для нового чата.
- `project-onboarding` - вход в проект через минимальный актуальный контекст.
- `scope-guard` - ограничение широких запросов.
- `change-review` - проверка изменений без перечитывания всего проекта.

## Быстрый старт

1. Скопируйте нужный навык из `skills/` в вашу папку Codex skills.
2. В новом чате напишите, например:

```text
Используй навык context-economy.
```

3. Для проекта добавьте папку `00_Context` из `project-template`.
4. Заполните `CURRENT_STATE.md`, `NEXT_WORK.md`, `PROJECT_MAP.md`.
